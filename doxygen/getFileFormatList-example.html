getFileFormatList

Get all formats which should be able to read/write the given format stack optionally using the given dialect.

Get all formats which should be able to read/write the given format stack optionally using the given dialect.

Parameters
format_stackthe maximum format stack to look for.

({"dcm","tar","gz"}, "") will look for plugins that cat read "*.dcm.tar.gz" files. As this will likely fail it will look for plugins that can handle *.tar.gz files, and if that fails as well, for *.gz files. The failed formats will be forwarded to the found plugin to deal with (the gz-reader will have to look for a *.tar reader, which will have to look for *.dcm).
